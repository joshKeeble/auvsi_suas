Operator Classification and Localization

Retrieve the images from the UAV during flight stored on the USB stick or wirelessly copy them over which is faster as it does not require the disassebly of the payload bay. The simplest way is over secure copy:

scp pi@192.168.1.66:path_to_directory path_on_ground_station_laptop

Remember to change this command in case you change the ip address of the pi to correspond with the new address.

Steps for Creating Objects

After finding an image with the target in view, use the cropping tool below to isolate the target so that it fills at least 25% of the frame

If the frame won't resize, select "Free" on the right side tool panel.

After you have cropped the image, you need to press JPG and the cropped image will pop up.

Save that image to the folder where objects are saved by deafult, /auvsi_suas/objects

This is important as it is specifically shown in documentation.

Press "Save Object" to create an object file with the data entered.

Keep track of how many objects are in the folder and label the JPG on which one it is, [1,2,3..].
